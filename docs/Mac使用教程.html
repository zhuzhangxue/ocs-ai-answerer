🍎 OCS AI 答题助手 — Mac 教程

手动部署 · 终端操作 · 约 10 分钟
⚠️ Mac 没有一键启动脚本，需要手动在终端操作。Windows 用户请直接看使用教程.html，不用看这个。
1

检查或安装 Python

打开「终端」（Terminal），复制下面整段命令粘贴过去按回车：

python3 --version

如果显示 Python 3.8+ 说明已有，跳过这步。
如果报错「command not found」，说明没装 Python，运行下面命令安装：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install python@3.11

第一条命令装 Homebrew（Mac 的软件管理器），第二条命令装 Python。过程中会要你输入电脑密码，正常现象。

2

填 API Key

把下载的 zip 解压，找到 keys.txt，用文本编辑打开，把里面的占位符替换成你的真实 Key：

DEEPSEEK_API_KEY=sk-你的DeepSeek密钥    ← 必配
DOUBAO_API_KEY=ark-你的豆包密钥        ← 必配
GLM_API_KEY=你的智谱GLM密钥           ← 可选

保存后关闭。

3

进入项目目录 + 装依赖

在终端里运行下面命令，一行一行复制粘贴执行：

cd 桌面/OCS-AI-Answerer-一键包
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
💡 第一行什么意思？cd 就是"进入文件夹"的意思。桌面/OCS-AI-Answerer-一键包 是你解压后文件夹的路径。如果你解压到了别的地方（比如「下载」里），就把第一行改成 cd 下载/OCS-AI-Answerer-一键包。

等待安装完成（1-3 分钟）。

4

启动服务

还是在终端里，运行：

python3 ocs_ai_answerer_advanced.py

看到服务启动信息后，最小化终端就行，不要关掉（关了服务就停了）。

✅ 终端显示服务运行中 = 后端就绪
5

浏览器部署

这部分和 Windows 完全一样，看 使用教程.html 的第四、五阶段：

  1. 装 Tampermonkey 扩展（Safari/Chrome 都行）
  2. 装 OCS 网课助手脚本
  3. 导入 ocs_config.json 题库配置
6

开始答题

登录超星/学习通，点 OCS 悬浮窗开始答题即可。终端不要关，关了服务就停了。

下次使用先开终端，cd 到项目目录，执行：

source venv/bin/activate && python3 ocs_ai_answerer_advanced.py
💡 Mac 和 Windows 的区别：除了启动方式不一样，其他（OCS 脚本、AI 模型、正确率、API 费用）完全一样。
