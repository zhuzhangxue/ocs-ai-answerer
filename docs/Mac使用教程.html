🍎 OCS AI 答题助手 — Mac 教程

双击启动 · 约 5 分钟
💡 Mac 启动脚本都在 scripts/mac/ 文件夹里，Windows 的脚本在 scripts/windows/，互不干扰。
1

填 API Key

把下载的 zip 解压，找到 keys.txt，用文本编辑打开，把里面的占位符替换成你的真实 Key：

DEEPSEEK_API_KEY=sk-你的DeepSeek密钥    ← 必配
DOUBAO_API_KEY=ark-你的豆包密钥        ← 必配
GLM_API_KEY=你的智谱GLM密钥           ← 可选

保存后关闭。

2

双击一键启动（仅首次）

打开 scripts/mac/ 文件夹，找到 一键启动.command，双击它。

首次运行会自动：检查 Python → 装依赖 → 配置 Key → 模型自检 → 启动服务

💡 如果提示 「 permission denied」，打开终端运行：
chmod +x scripts/mac/一键启动.command
然后就能正常双击了。
✅ 看到服务地址 http://127.0.0.1:5000 = 启动成功
3

Mac 版启动脚本一览

scripts/mac/ 文件夹里有三个脚本：

脚本用途
一键启动.command首次配置 + 日常启动（装依赖、填 Key、启动）
启动服务.command后台启动服务（无窗口）
停止服务.command停止答题服务

Windows 的 脚本/windows/ 里也有对应的 一键启动.bat、启动服务.vbs、停止服务.bat，但那是 Windows 用的，Mac 不用管。

4

浏览器部署

这部分和 Windows 完全一样，看 使用教程.html 的第四、五阶段：

  1. 装 Tampermonkey 扩展（Safari/Chrome 都行）
  2. 装 OCS 网课助手脚本
  3. 导入 ocs_config.json 题库配置
5

浏览器部署

这部分和 Windows 完全一样，看 使用教程.html 的第四、五阶段：

  1. 装 Tampermonkey 扩展（Safari/Chrome 都行）
  2. 装 OCS 网课助手脚本
  3. 导入 ocs_config.json 题库配置
6

开始答题

登录超星/学习通，点 OCS 悬浮窗开始答题即可。终端不要关，关了服务就停了。

下次使用先开终端，cd 到项目目录，执行：

source venv/bin/activate && python3 ocs_ai_answerer_advanced.py
💡 Mac 和 Windows 的区别：除了启动方式不一样，其他（OCS 脚本、AI 模型、正确率、API 费用）完全一样。
