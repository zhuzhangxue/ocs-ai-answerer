📥 安装浏览器扩展 (脚本管理器)

浏览器需要装一个"脚本管理器"扩展,才能运行 OCS 油猴脚本。

推荐：脚本猫 (ScriptCat) —— 中文好用、稳定

Chrome 商店一键安装 Edge 商店一键安装

备选：Tampermonkey (油猴)

访问官网
💡 如果用 Edge 浏览器,推荐用脚本猫（轻量、专为中文用户设计）
💡 如果商店打不开,可以去 scriptcat.org 找离线包
2

📜 安装 OCS 网课助手脚本

扩展装好后,去 OCS 官网一键安装网课助手脚本:

打开 OCS 安装页面

在 OCS 官网点"安装到脚本管理器"按钮,扩展会自动接管。

💡 OCS 官方支持超星学习通、智慧树、职教云等所有主流网课平台
💡 安装后,打开任意网课平台页面,右下角应出现 OCS 悬浮窗
3

🔧 导入 AI 题库配置

打开任意网课平台的考试页面 → OCS 悬浮窗会出现在右下角

  1. 点 OCS 悬浮窗的 「通用」 标签
  2. 点 「全局设置」 齿轮按钮
  3. 找到 「题库配置」 → 点 「点击进入配置」
  4. 点 「新建配置」 或编辑现有
  5. 把 ocs_config.json 的内容全部复制粘贴进去
  6. 点 「解析并保存」（这一步要联网,因为要解析 ocs_config.json）
✅ 看到 "url": "http://127.0.0.1:5000/api/answer" 字段就说明指向了本地服务
4

🎉 登录网课平台,开始答题

现在打开任意网课平台（学习通/智慧树/职教云等）:

  1. 用你的账号登录
  2. 进入任意 章节测试 / 作业 / 考试 页面
  3. OCS 悬浮窗应该自动开始搜题
  4. 看到 「AI 智能答题」 标签 = 成功! ✅
⚠️ 如果悬浮窗没出现:
  • 检查扩展是否启用 (浏览器右上角扩展图标)
  • 刷新页面 (F5)
  • 点扩展图标 → 检查 OCS 脚本是否已启用

🔑 第一次使用: 填 API Key

解压包里有个 keys.txt,用记事本打开,把你的 API key 填进去:

DEEPSEEK_API_KEY=sk-你的真实密钥
DOUBAO_API_KEY=ark-你的真实密钥

填好保存,再双击 一键启动.bat,程序会自动读取 keys.txt,完全不用手动改 .env。

以后再启动,key 已经存在 .env 里了,跳过这一步。

💡 怎么拿 key:
• DeepSeek: platform.deepseek.com/api_keys
• 豆包: console.volcengine.com/ark → API Key 管理
