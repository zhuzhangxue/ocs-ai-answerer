🔑 第二阶段：获取 API Key 3 分钟
💡 什么是 API Key？它是你调用 AI 大模型的"通行证"。按需求配置，DeepSeek 和豆包是核心必配，GLM 和 Qwen 是图片题的降级补充，可跳过。
1

注册 DeepSeek 并获取 Key 必配

DeepSeek 负责"思考答题"，是整个项目的大脑，必须配置。

打开 DeepSeek 平台 →
  1. 点击右上角「注册」(支持手机号/微信)
  2. 登录后点击左栏「API Keys」
  3. 点击「创建 API Key」→ 名字随便填(如 "OCS")
  4. 立即复制显示的 key（形如 sk-xxxxxxxxxxxxxxxx）
⚠️ Key 只显示一次！必须立刻复制保存到记事本
💰 费用: 充值 1 元就够用很久(V4 极便宜,具体价格以 DeepSeek 价格页 为准)
2

注册智谱并获取 GLM Key 可选

GLM-4.6V-Flash 是免费的图像理解模型，作为识图首选模型。失败时会自动降级到豆包。不配也可以，图片题会自动用豆包。

打开智谱 AI 开放平台 →
  1. 用手机号注册登录
  2. 进入「API Keys」页面
  3. 点击「添加 API Key」
  4. 立即复制Key（形如 xxxxxxxx.xxxxxxxxxxxxxxxx）
💰 费用: GLM-4.6V-Flash 目前免费，无需充值
3

注册火山方舟并获取豆包 Key 必配

豆包负责"看图认字"(OCR)，图片题必须依赖它，必须配置。

打开火山方舟 →
  1. 用抖音/手机号注册登录
  2. 点击左侧栏目「推理」
  3. 进入「在线推理」页面
  4. 点击「快速入门」
  5. 在快速入门页点击「创建 API Key」
  6. 勾选所有模型权限 → 创建
  7. 立即复制Key(形如 ark-xxxxxxxx-xxxxxxxx-xxxxxxxx)
💰 费用: 豆包 seed 系列为付费 API，有试用额度，非永久免费。超额后按量计费。
⚠️ 重要：免费额度用完后如何继续使用？
火山方舟默认开启了「安心体验」模式，免费额度耗尽后会自动暂停服务，不会消耗余额。
如果你想额度用完后继续用余额付费调用，请前往：
火山方舟控制台 → 开通管理 → 找到「安心体验」→ 点击「关闭」
关掉后才能正常扣余额继续使用。
4

注册阿里百炼并获取 Qwen Key 可跳过

Qwen VL Flash / Qwen3.7-Plus 是通义千问视觉模型，在云端降级链最末端。不配完全不影响使用，建议跳过。

查看百炼文档 →
  1. 用阿里云/支付宝账号登录
  2. 进入模型广场 → 开通 Qwen VL Flash
  3. 创建「API Key」（DashScope API Key）
  4. 获取「工作空间 Base URL」（百炼平台 > 模型调用 > OpenAI 兼容地址，每个用户不同，包含工作空间 ID）
⚠️ QWEN_BASE_URL 不是固定的 https://dashscope.aliyuncs.com/compatible-mode/v1，必须在百炼控制台获取您工作空间专属的地址。
💰 费用: Qwen VL Flash 和 3.7-Plus 均为付费 API，有试用额度。
5

在豆包平台开通模型 必需

开通模型后才能调用,操作是免费的。提供两种方式,选一种即可：

方式 A：快速入门入口(推荐)
  1. 还是刚才的「快速入门」页面
  2. 往下翻,找到「Step2 快速接入测试」模块
  3. 在模型下拉框里选 doubao-seed-2-0-mini
  4. 点旁边的「开通」按钮
  5. (可选)再选 doubao-seed-2-1-turbo 和 doubao-seed-2-1-pro 也开通
方式 B：模型广场入口
  1. 顶部菜单点「模型广场」
  2. 搜索 doubao-seed 找到 seed 系列
  3. 点右上角「API 接入」按钮
  4. 会跳到「Step2 快速接入测试」页面
  5. 在模型下拉里选 doubao-seed-2-0-mini → 开通
  6. 同样方法开通 turbo 和 pro

填入 keys.txt 关键

把里面的占位符替换成你的真实 key（GLM 和 DASHSCOPE 可跳过不配）：

# 改成你的真实 key（删除 <> 部分,只保留 key 本身）
DEEPSEEK_API_KEY=sk-你刚才复制的 DeepSeek key    ← 必配
DOUBAO_API_KEY=ark-你刚才复制的火山方舟 key      ← 必配

❓ 常见问题 FAQ
Q1: 双击 bat 后,cmd 窗口一闪而过?
可能是编码问题。右键 bat → 编辑,把第 2 行 chcp 65001 改成 chcp 936,或者用 Windows Terminal 运行。
Q2: 步骤 1 提示「未检测到 Python」?
Python 没装或者没勾「Add to PATH」。重装 Python 时务必勾上 Add Python to PATH 选项。
Q3: 步骤 3 依赖安装失败?
国内网络问题。先在 cmd 运行：
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
再重新运行 bat。
Q4: 步骤 4 检测不到 key?
检查 keys.txt 格式：
① 不能有空格（KEY=值，不是 KEY = 值）
② DeepSeek key 必须 sk- 开头，豆包 key 必须 ark- 开头
③ 不能有引号
④ 不能用中文标点
GLM key 非必需，没有的话图片题直接用豆包，不影响使用
Q5: 浏览器打开 127.0.0.1:5000 提示「无法连接」?
服务没起来。在 cmd 窗口里看有没有错误,或者检查 5000 端口是否被别的程序占用。
Q6: OCS 导入题库报「Spread syntax」错?
OCS 脚本版本太老(低于 4.7.21)。去 OCS 官网更新脚本到最新版。
Q7: 答题时显示「此题未完成」?
常见原因(按概率排序)：
① OCS 端用「相似匹配」→ 改成「精确匹配」
② 答案格式不匹配 → 单选答字母而非文字
③ 图片题 GLM/豆包调用失败 → 检查 .env 里 key 是否有效
④ 超时 → OCS 设置里把题库耗时调到 60s
Q8: 本地 OCR 是什么？需要开启吗？

项目内置了 PP-OCRv6 本地 OCR，可以在你电脑上直接识别图片文字，不消耗任何 API 额度。

默认关闭，因为目前的网课图片题大多是数学公式、函数图、表格等，本地 OCR 对这类内容的识别率很低，不如直接走云端视觉模型（GLM / 豆包）准确。

如果你确定题目只有纯文字截图（没有公式图表），可以尝试开启：

  1. 打开 .env 文件，找到 ENABLE_LOCAL_OCR=false 改成 true
  2. 重启服务生效

开启后流程图：本地 OCR 提取文字 → 达标则直接答题（省去调用云端视觉模型）→ 不达标则自动降级到云端视觉链。

Q9: 怎么添加新模型(如 Kimi)？
打开 docs/添加新模型教程.html 里有完整步骤。或者直接告诉我「加 Kimi, key 是 xxx」,我帮你加。
Q10: 换电脑后怎么迁移?
压缩整个 OCS-AI-Answerer-一键包 文件夹(含 venv) 拷到新电脑,改 keys.txt,双击 bat 即可。venv 目录可以删掉重新创建(小电脑空间不够时)。
Q11: 想换个模型(如 Flash 改 Pro)？
打开 docs/配置教程.html 里有详细步骤。或者告诉我「单选切换到 Pro」,我帮你改。
Q12: 守护程序怎么用？每次开机都要重新双击吗？

双击一次 启动守护.vbs 后它就在后台持续运行了（每30秒检测一次，几乎零CPU）。

想让它开机自动运行：把 启动守护.vbs 的快捷方式放到 启动 文件夹即可（按 Win+R → 输入 shell:startup → 把快捷方式粘贴进去）。

想停止守护：双击 停止服务.bat，它会同时停掉答题服务和守护程序。
